1.2.5. patsy.EvalEnvironment¶

class patsy.EvalEnvironment(namespaces, flags=0)[source]¶

Represents a Python execution environment.

Encapsulates a namespace for variable lookup and set of __future__ flags.

__init__(namespaces, flags=0)[source]¶

1.2.5.1. Methods¶

__init__(namespaces[, flags])

eval(expr[, source_name, inner_namespace]) Evaluate some Python code in the encapsulated environment.
subset(names) Creates a new, flat EvalEnvironment that contains only the variables specified.
with_outer_namespace(outer_namespace) Return a new EvalEnvironment with an extra namespace added.
